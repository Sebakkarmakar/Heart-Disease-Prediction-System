## 📖 Project Overview
The Heart Disease Prediction System is a machine learning–based application that predicts whether a person is likely to have heart disease based on medical parameters.
The system uses a Logistic Regression model trained on a heart disease dataset and provides predictions through an interactive Streamlit web interface.
## 🎯 Objectives
- Predict heart disease accurately using machine learning
- Provide a user-friendly interface for medical data input
- Assist in early detection and awareness
- Demonstrate end-to-end ML project implementation
