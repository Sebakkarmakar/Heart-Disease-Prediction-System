❤️ Heart Disease Prediction System
## 📖 Project Overview
The Heart Disease Prediction System is a machine learning–based application that predicts whether a person is likely to have heart disease based on medical parameters.
The system uses a Logistic Regression model trained on a heart disease dataset and provides predictions through an interactive Streamlit web interface.
## 🎯 Objectives
- Predict heart disease accurately using machine learning
- Provide a user-friendly interface for medical data input
- Assist in early detection and awareness
- Demonstrate end-to-end ML project implementation.
## 🧠 Machine Learning Model
- Algorithm Used: Logistic Regression
- Type: Binary Classification
- Target Variable:
- 0 → No Heart Disease
- 1 → Heart Disease
## 📊 Dataset Information
- Source: Heart Disease Dataset
- Total Records: 271
- Features Used:
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Max Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- Number of Major Vessels
- Thalassemia
## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Pickle
## 🖥️ Frontend
- Built using Streamlit
- Interactive input fields with medical explanations
- Real-time prediction output
- User-friendly UI for non-technical users
## 📈 Model Performance
- Accuracy: ~88.5%
- Precision: ~0.89
- Recall: ~0.88
- F1-score: ~0.88
- Balanced dataset (no class imbalance)
